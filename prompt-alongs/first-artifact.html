An interactive tool where you enter any two options, define the criteria that matter to you, score each one, and get a weighted recommendation. You'll type one prompt and get it back as a working artifact.

Open claude.ai in your browser. Start a new conversation. Then copy the prompt below and paste it in.

The tool uses two numbers for each criterion:

Weight = how much this criterion matters to you. "Salary" might be high-weight; "office snacks" might be low-weight. The tool might use sliders, dropdowns (low/medium/high), or numbers — whatever Claude decides looks best.

Score = how well each option performs on that criterion, from 1 (terrible) to 10 (amazing). You score each option separately.

The tool multiplies weight × score for each criterion, then adds them up. The option with the higher total wins. It's the same math a spreadsheet would do, but the tool makes it visual and easy.

Build me an interactive decision-making tool as an artifact. I want to compare two options side by side using weighted criteria. The tool should: 1. Let me enter two options I'm deciding between (e.g., "Take the new job" vs "Stay at current job") 2. Let me add criteria that matter to me (e.g., salary, work-life balance, growth potential) 3. Let me weight each criterion by importance (drag a slider or pick high/medium/low) 4. Let me score each option on each criterion (1-10 scale) 5. Calculate a weighted total score for each option 6. Show a clear visual comparison — bar charts or some kind of visual 7. Give a recommendation based on the scores Start with some example criteria pre-filled so I can see how it works right away. Make it look polished and modern — clean design, good use of color, satisfying to interact with. I want it to feel like a real app, not a homework assignment.

Your tool works, but it could be better. The most powerful thing about working with AI isn't getting a perfect first result — it's how fast you can iterate. Try any of these:

"Add a brief summary at the top that explains what this tool does and how to use it in 2-3 sentences."
"Improve the visual design — add subtle gradients, better spacing, and a more modern color palette. Make it feel premium."
"Support comparing 3 or more options, not just two. Let me add and remove options dynamically."
"Add an export button that copies my comparison results as formatted text I can paste into a document or email."
"Add a reset button that clears all scores but keeps my criteria and options, plus a full reset that clears everything."
